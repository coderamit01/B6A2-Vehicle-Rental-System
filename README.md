# Vehicle Rental System

**Live URL:** https://vehicle-rental-system-silk.vercel.app/

## Features

- User registration and login
- Admin can add, update, and delete vehicles
- View all vehicles and single vehicle details (public access)
- Vehicle booking system
- Multiple booking management
- Update, cancel, and return bookings

## Technology Stack

- Node.js
- Express.js
- PostgreSQL
- bcrypt (password security)
- jsonwebtoken (authentication)

## Setup & Usage Instructions

1. Clone the repository:
```bash
git clone https://github.com/coderamit01/B6A2-Vehicle-Rental-System.git
```

2. Open VS Code and run:
```bash
npm install
```

3. Create a `.env` file and add:
```env
SECRET_KEY=your_secret_key
CONNECTION_STRING=your_postgresql_connection
PORT=your_port_number
```

4. Run the server:
```bash
npm run dev
```
